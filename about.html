Project Overview

This application predicts the median house value for a California block-group using a feed-forward Artificial Neural Network (ANN). Inputs include geographic coordinates and housing/demographic attributes (e.g., rooms, households, population, median income, and ocean proximity).

  • Framework: Flask (Python)
  • Model: Keras/TensorFlow ANN
  • Preprocessing: MinMaxScaler
  • Artifacts: model_ann.h5, scaler.pkl
  • UI: Bootstrap 5, responsive
  • Serving: Local Flask server
How It Works
  1. User enters features on the form (Home page).
  2. Server scales inputs with the trained MinMaxScaler.
  3. ANN predicts price; the result is rendered back to the UI.
Model & Data

Trained on the well-known California Housing dataset (block-group level). Key features: longitude, latitude, housing_median_age, total_rooms, total_bedrooms, population, households, median_income (×$10k), and ocean_proximity.

  • Loss: Mean Squared Error (MSE)
  • Metrics: MAE / R² (in notebook)
  • Encoder: numeric map for ocean_proximity
Limitations
  • Predictions are illustrative; not for financial advice.
  • Generalization is bound to training data distribution.
